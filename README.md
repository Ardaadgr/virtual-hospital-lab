# Virtual Hospital Lab

A digital twin hospital network simulation platform.

## Features
- Containerized hospital services
- Network segmentation (VLAN simulation)
- HBYS / PACS / STAFF systems
- Security monitoring layer (future)
- AI anomaly detection (future)

## Tech Stack
- Docker
- FastAPI
- Linux networking
- Python

## Goal
To simulate a real hospital IT infrastructure for learning,
cybersecurity testing and AI-based anomaly detection.
